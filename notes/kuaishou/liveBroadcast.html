话术 #

话术流程框架：拉新（活动，痛点，互动）-> 讲款（卖点，场景，信任，互动） -> 保障（信任）-> 开款（转场，活动，开款，指导，留存） -> 促单（促单，互动，信任） -> 转场

主播不是销售员，是美好生活的规划师，我们在规划用户的生活，规划的是用户拥有这件商品之后的生活，卖的是生活场景

活动话术

活动话术能创造更好的销售氛围，提高直播间的转化率

一.活动话术最主要的是活动主题
以店铺为主题 - 店铺周年庆，销量，涨粉
以人为主题 - 生日，结婚，获奖，带货榜
以账号为主题 - 账号一周年
以节庆为主 - 616，818，双11


二.如何设计好活动话术？
1.先讲活动主题，再讲活动形式
2.低价产品突出价格优势，价格就是活动的核心
3.高价产品突出价值优势，价值不到，价格不报

卖点就是痛点的反面，是痛点的解决方案。挑重点的3个卖点讲就可以，讲太多反而没有重点了

塑造产品价值

1.成本拆解-比如卖一件衣服69元，衣服上10个扣子，每个扣子都5块钱，这10个扣子都值50了，那这衣服卖69还贵嘛

2.痛点击穿

3.场景代入-产品拿到手之后，用在什么样的场景下。本来人家不需要这件衣服，但是你跟人家一说，姐妹们，今天这件衣服你们必入，新年到了，你要不要参加同学聚会，你们好多人我知道啊，都已经约好了同学聚会了，但是一到室内大家是不是都要脱衣服，别人一脱，里面就是一身黑色打底衫，特别枯燥乏味，但是你呢，你一脱，你看你这衣服领口还有绣花，肩膀上还有绣花，胸口上还有绣花，但我们家领口做的很好，就是黑色的，所以特别适合配大衣，又特别适合配羽绒服，但是还别有乾坤，对吧，到了室内你把衣服一脱，别人就是一身黑一身白的，但你呢，你的衣服上布灵布灵的，闪亮亮的，你就是人群中最亮那个仔

4.对比塑造-对比是最有效的方式，你能把两个产品放在一起对比，你就能迅速出单，因为人是视觉动物

5.信任背书-比如多少网红在用啊。还可以搞一些代表性的融入，比如，我跟你们说啊，你们这件衣服必买，为什么，但凡懂点时尚的，都一定会买它。在比如，快手上80%的宝妈都会来我这儿买这个东西

6.赠品塑造-在送赠品的时候，我今天这个东西卖139，但是我今天会给你送一个89的产品，把这个赠品也挂在小黄车，你们看我这个产品平时卖的时候，是不是卖89，所以我今天给你一个139的产品，但是我还同时送你价值89的东西

7.真诚是必杀技，好的坏的都可以说-不要光说你们家好的地方，也要说说你们家缺点的。比如，你们放心，我们家直播间从来不玩套路。在比如，衣服扣子5块钱一个，成本比较高，所以没有备用扣，但是如果你们扣子要是坏了的话，你们不会缝，拿出去外面缝的话，我可以给你们出工本费的

直播脚本 #

流量

1.直播间自然流量
必须有低价引流品，低价引流品点击率需达10%，如果达不到10%，拉流量是很困难的，用它憋单也很困难。如果不想用低价引流品，那需选爆品，这个爆品点击率需达15%以上，才能靠爆品拉起自然流量，爆款可设置区间价或者AB链，来憋一憋人气。

2.短视频流量
通过直播时，将想测的品，放在小黄车6，7，8号链接，挂个3-5天，基本可以测出来哪个品点击率最高，如果这个品点击率大于10%，可以用它来拍视频，8-15秒为主，多分镜头，每3秒一个分镜，配合热门音乐，这种素材适合金牛审核。如果是引流短视频，选题大于内容大于剪辑，可以拍些测评视频和口播类负向话题，例如告诉大家一个秘密，99%的和田玉都是假货。在视频的发布时间上，如果你的基础播放量不高，开播前1个小时或者1个半小时发就可以，如果播放量比较高，有20万，30万的播放量，那开播前一天晚上或者半天前发就可以，给视频充分进入下一个流量池的时间，如果能双频共振，可以突破流量层级。

3.金牛付费流量
用金牛走付费流量，必须要有爆款。

如果客单价比较低，人群受众比较广，比如尾货，日用品，可以通过快速过款密集出单或者憋单把直播间拉起来人气的形式，获得自然流量的，自然流量的占比最好占到70%，付费占30%，如果付费占比过高，你卖一个低客单价的产品，容易不挣钱或亏钱。

如果是中高客单价，比如女装，文玩，建议用短视频流量去做，而且短视频介入的流量要占比到整个直播的30%-60%是最好的。

如果是高客单价的产品，有成本优势，比如，珠宝，品牌产品，一般500以上算是高客单价，肯定要用付费流量，付费流量占整个直播的30%-50%。

低高中-低高中-低高中，比如19.9，59.9，29.9

联想顺序（袜子，睡衣）
美妆-护肤顺序（洁面，补水，精华，霜）

②跨界

根据营销目的排款，比如今天搞大厂，福利品得放放足，3号面膜是个主推款，那可以，1号1元化妆包，2号9.9元吹风机，3号178元面膜

直播强化

1.促单转化话术标准化
比价话术-这款在某宝多少钱，或者在专柜多少钱，今天我们直播间不要，只要多少钱
服务促单-7天无理由，运费险，质检证书，全国复检，精致礼盒等等
2.互动技巧标准化-每次商品展示要一次互动，开价前要一波互动
3.销售氛围营造-水军互动，喊库存，卡库存返场
4.红包，福利购配合
5.做专业话术，特别是卖一些贵的东西，或跟中国传统文化相关的一些东西，比如玉石翡翠，搞一些专业术语出来，让人觉得很有文化底蕴，产品值这个钱

营销玩法

解决的是直播间的停留，互动，商品点击的问题
1.排款-要有个低价引流款，低价引流款会起到提高互动，增加用户停留的作用，可以设置已抢光，并把价格打开，比如设置个1块钱的大肚杯，这样可以帮你修一下留人数据。这个低价引流款也可以在场中用，比如你在直播中是走这种脚本的，微憋单5分钟，2分钟促单，7分钟开始放单，这种风格的，你接下来又去介绍其他产品了，这个产品介绍了5分钟了7分钟了，也不出单，这种情况下，没有成交密度了，这中间就可能有7分钟到10几分钟的断档，就会导致流量的下滑，这时可以拿低价引流款去补补密度，补补数据，偷放个三五单

2.红包-红包有普通红包，互动红包，粉丝团红包，分享红包。开播时用互动红包，设置100快币，10个人，10分钟挂在哪。设置一个互动口令，比如接下来是要上福利品，福利品是个包包，那在口令里可以设置抽包包，去做开播的留人和互动，就是告诉大家你接下来要上一个什么福利品，让大家知道，现在正在抽奖，还是在搞什么活动。红包不要从头到尾用，开播时可以发，即将上架一个爆款的时候可以设置一个指向型口令，比如，要裙子，要口红，去帮你做转款氛围和转款铺垫。再一个是配合主播返场的时候设置口令，比如，没抢到，加一波，来做返场销售氛围。

3.抽奖-互动高，停留高，增粉高，增粉丝团高，抽奖是最好的，增加浅层指标的工具。抽奖工具有好运来，幸运助手，福利购

4.打爆款-爆款的互动一般，停留一般，但成交高，商品点击高。如果你平时能测到一款，曝光点击率15%，成交点击率15%以上的产品，那适合循环打爆款，只打一个品。

5.做轻内容-增加互动和停留。比如做服装的，可以走走秀。卖食品的，可以做一些美食的制作。

6.购物车排序+卖点-小黄车前5个链接放过往3场到5场点击率较高的商品，按价格低中高排序更友好。点击率特别高的可以设置已抢光，做微憋单。还可以前5个链接都设置1元，可以帮你拉停留时长，增加商品点击，不过这种方式，针对的是人群比较泛，客单价低的商品。

7.复合链接，区间价-比如卖童装，这个童装最后是打算卖49的，在一开播的时候，可以拿个牌子，或者在小黄车1号链接设置一个9块钱，让别人误以为便宜，可以增加停留，增加点击量，最终上架去卖的时候，换成真链接，换成49元。

AB链的进阶玩法-总共会有3个链接，比如卖大肚杯，A链接可以在憋单环节设置3.9元，看人气憋到500人1000人了，快要放单之前，换成B链接，设置为239元，这个时候是用B链接洗一下人群，最后上车的时候，上真的链接89元，去放单。

话术留人，憋单节奏 #

留人话术

我们的刘二呵来了，我们的许知道来了，我们的小雨来了，我们的小果来了，你们四个来了，恭喜你们，你们4个来的正是时候，我1分钟之内刚好要给你们抽一个干货大礼包，这个干货大礼包里有整整9本的知识干货，电商人拿着直接就能用，而我们今天新号开播，我们现在直播间就你们四个人，所以待会你们中奖的概率高达四分之一，今天抽奖的方式超级简单，就是给我点一下关注，亮一下灯牌，我取第三个亮灯牌的同学，直接带走干货大礼包，包邮到家，而且大家如果3分钟之内没有中奖，我3分钟之内再给大家开一轮。

憋单的4个阶段

1.人少阶段---人少要一对一，推心置腹，点对点，用真假真话术

真假真话术
许知道，欢迎你来到我的直播间，我看到你了宝贝，不管你今天在不在，我都要把今天的活动内容和你说一下，我看到这个叫美美的宝贝已经在我直播间呆了2天了，美美宝贝，你今天终于愿意给我点关注亮灯牌了宝贝美美，这样吧美美宝贝，我今天呢看到你很主动地给我点了关注，所以呢，我作为主播，我个人自掏腰包请你试穿一下我们的这件连衣裙好不好宝贝，你直接把你的身高体重待会儿和我们地客服沟通一下，直接给你安排到家，一分钱都不用出的，直接让你安排到家，你去试穿，好吧，许知道宝贝，如果你也想和美美宝贝一样地福利，你就扣个在，如果你不需要，咱们就把这个福利给别人了

2.聚人阶段---把整个节奏重复，内容不能重复，变着花样讲话术。话术穿插其他东西，表现真诚感，比如我直播间没有任何的套路，金杯银杯不如大家的口碑。聚人阶段就是通过话术把人做停留做互动。

3.拉人阶段---憋互动憋流速，主要看推流节点是怎么推的，如果你直播间能突破18个人，你接下来到25个人就很轻松，如果能把这25个人，稳定一分钟左右，那就能到30个人了，30个人就破了流量池了，30个人如果能有6个人互动，接下来就能上到42个人左右，如果42个人里给你点了2个关注，亮了5个粉丝灯牌，那很快就能突破70个人，正常42个人到70个人，5分钟之内就能拉到。这时你要让大家疯狂点击小黄车，去做你的浅层数据，增加小黄车权重。一般花40分钟，就能到70个人。

4.洗人阶段---在70个人以前，放个1-2单，快开快放，虽然1到2单不足以打标签，但已经能让系统捕捉到你能干嘛了，聚的是爆款人群。千万不要到100人在去放单，到100人会风控一下，因为你破流量池了，如果你还憋单不放单，有可能会罚款。

不同人气话术拆解-自然流憋单起号 #

0-30人在线，点对点破冷启

开播前15分钟，做互动，做到4%的评论，3%的转粉。如果4%的评论达到之后，没有推流，可能就是老粉互动，需要做拉新。如果前15分钟，人气数据做的不好，没有达到4%的评论，3%的转粉，这会该做电商数据了，可以用截流或鱼塘的方法去做一些电商数据。如果还不行，就投流。

1.抛钩子 - 要有福利，要有钩子产品，钩子画面和钩子话术，而且福利是一直要讲的，因为每分钟都有新人进来
2.为什么有福利 - 告诉人家今天是新号开播，所以有福利，活动原因讲了还不够，信任话术要去讲
3.说明互动的要求 - 是给评论，点关注，还是加灯牌
4.先点假人 - 如果始终要不到互动，让水军出来互动一下
5.再点真人

30-80人在线，关键词强互动

流量转化漏斗 #


曝光点击率

曝光点击率 = 观看人数 / 曝光人数

曝光点击率在20%-30%合格
低于10%，需要大调场景
在10%-20%的，小调场景
超过50%了，人群不精准，需优化人群标签

商品曝光率

商品曝光率 = 商品曝光 / 观看人数

商品曝光率在60%-80%合格

商品曝光率低的原因
1.没有开讲解
2.主播话术没有引导进小黄车
3.讲品枯燥，讲货时间太长
4.人群不精准
5.人货场，场不吸引人（3秒有效观看少），主播不招人喜欢
6.社交型直播间

商品点击率

商品点击率 = 商品点击 / 商品曝光

商品点击率在20%-30%合格

商品点击率低的原因
1.产品太贵-要塑品，说明产品有价值
2.不是刚需产品，觉得可要可不要-话术多讲痛点
3.链接上的过多-建议不超过12个品
4.商品链接（头图）太难看
5.排品不对
6.主播没有引导
7.话术没有紧迫感
8.中控没有弹窗

下单率

下单率 = 下单人数 / 商品点击

下单率在20%以上合格

下单率低的原因
1.福利不够
2.促单话术不给力
3.后顾之忧没讲好
4.没有让人觉得有后悔的余地

支付率

支付率 = 支付人数 / 下单人数

支付率在80%以上合格

成交转化率

成交转化率 = 支付人数 / 商品点击数

成交转化率在10%以上合格

各项指标期望值

成交人数 = 累计成交人数 / 累计观看人数 --- 期望在1%
成交转化率和流量转化漏斗的成交转化率相同

粉丝复购率 --- 期望在60%以下
人均观看时长 --- 1分钟合格，2分钟良好，3分钟优秀

千次优秀区间计算

累计成交金额 / 累计成交订单数 * 5 - 累计成交金额 / 累计成交订单数 * 10
千次决定人群精准度，和人气高低

老号重启 #

什么是老号

1.长时间不直播了
2.账号一直在播但永远处在30人以下的最低流量池
3.账号不盈利
4.磁力金牛跑不动
5.在两三年之内，陆陆续续涨过几万粉丝但是从来没有直播过

老号怎么救

1.提高活跃度
拍视频：日更1-3条段子，更5-7天。可以3条批量投放，也可以分开投放，前6条投粉条的点赞评论，中间6条投涨粉，最后几条投主页浏览，投24小时。另外开播前2个小时，将1到2条跑量好的视频，设置为置顶视频并投6小时主页浏览，投放金额为300到400元。

直播：坚持开直播，停播不得超过3天，超3天要降权。

2.救标签
选好爆品
做机制，破价5-8折，最好做5折
开播前15分钟，准备10个水军，做分钟级互动，拿到零星曝光。要用新账号做互动，不要拿以前在直播间聊过的账号去聊。拉新的原理是拉到新的人，官方觉得你能承接新的人，才会给你新的自然流量，所以要用新账号去做互动。这个人跟你互动5次以上，已经算你直播间的老互动了，就拉不了新了。

3.救流速
PC金牛，建1-3条商品购买计划，投作品进直播间，如果商品购买计划拉不进人，那就配合1条商品点击计划，帮忙做下放量，强拉下流量，刺激下账号的推流，注意做点状加热（消耗30，50元关闭，10-15分钟，再打开）

移动金牛，开播前40分钟，投三单商品购买，每单100元，选择自定义投放，开播一小时后，投ROI，选择智能投放

思路

开播前发段子，3-5天内发30个段子，每天6个或者10个

如何突破流量池 #

新号前十场直播，播到4个小时以上的，流量池给的比较好，标签打的也快

TIP

在直播的过程中，在快手小店电脑端，打开数据大屏，在每个数据下方都会有一行红字，显示超过同行，同等级多少，超过75%以上是优秀，但是超过同行，百分之七八十，不足以突破流量层级，至少达到超过同行同等级90%以上，才有机会突破流量层级，即便超过同行同等级90%以上，也不是立刻能破流量层级的，首先需要维稳，至少7天以上，或者最好每天销售额能做递增，注意不能做递减

做大场，可以更快的突破流量层级，做一波大的投流，拉一下直播时长，销售额做到平常的5倍到10倍，会发现在大场过后，流量层级是破了的

破流量层级最快的方式就是做憋单，突破流量层级的关键，除了出单外，是做浅层数据，突破在线阈值。比如直播中，发现直接间人数46个人，你要知道下一个流量层级的节点是50，你需要冲节点，拿能憋起来流量的产品也叫福利品，去拉数据，憋互动，憋停留时长。
