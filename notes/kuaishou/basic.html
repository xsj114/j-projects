快手基础 #

账号权重 #

获得更高的账号权重

活跃度-坚持发布短视频，定时定点的直播
贡献度-收到很多打赏，卖了很多货，帮助平台把人留住

如何判断账号权重

作品的平均播放量就是你的权重
直播间的平均在线人气就是你的权重

推流机制 #

标签推流 #

标签分类描述
喜好标签作为粉丝的标签
作品标签如果是非挂车作品，系统是根据卡关键词来打的标签
如果是挂车作品，系统是根据销量来打标签的
直播标签电商直播间是根据销量来打标签的

标签四要素

类目，性别，年龄，消费层级

TIP

从大数据中找出，账号中喜好标签上有你自己作品标签的人群

赛马机制 #

赛马机制四要素

同行，同等级，同时段，同消费层级

赛马需要做的数据

人气数据-关注3%，加团1%，评论3%-5%
电商数据-成交越多，模型越稳，流量越稳定。千次越高，平台推流越精准。

对应推送 #

TIP

同城表现好，推同城。关注表现好推关注。付费数据好，推付费。整体数据优秀，推自然流量。

账号自检 #

判断方式说明
搜索账号用其他的账号去搜索你账号的名字，如果很顺利的把账号找到，并且打开进入，那就没有问题
查看私信定期去关注官方发给你的私信，比如快手管理员
账号诊断工具快手管理员-个人主页-帐号诊断工具
快手管理员-对话框-私信菜单栏-帐号诊断
快手管理员-发送“违规查询”-违规查询
侧边栏-创作者中心-帐号诊断
侧边栏-快手客服-帐号诊断
开播页-主播中心-违规查询
购买粉条给每条短视频买一个最低消费的作品推广，如果你的这条短视频违规的话，会明确的在快手粉条里发一条私信告诉你什么原因,另外缺量退款的情况下，不是违规
金牛不消耗金牛买了完全不消耗
流量异常日常流量都在三五千，忽然一条短视频只有几百的播放量了
缺少同城你的直播一定是可以给你曝光在同城的,除非你违规了
