粉条投放思路 #

2024年粉条投放思路

1.投放顺序
播放量-涨粉量-主页浏览量

2.人群
相似达人-智能优选/自定义
注意：人群覆盖要达到1千万以上。投相似达人如果人群比较少，一种办法是拉时长去投放，原先是2小时看效果，可以投12小时或24小时，一种办法是如果你不着急打标签，只要播放量，可以直接打自定义

3.怎么看数据是否好
封面点击率大于10%，5s完播率大于50%，整体完播率大于20%，这种作品爆的几率比较大


投放步骤
1.针对相似达人，小金额投放测试看数据，投放时长2小时
2.作品右下角分享里找到数据，去分析数据是否需要追投
3.如果可以追投，可以大金额，多批次，拉时长，组合投放
大金额---原来是18，28的，现在可以100，200
多批次---就是继续追投
拉时长---正常18，28块钱投2个小时，如果是100，200块钱，可以投12小时，24小时
组合投放---如果数据跑的不错，追投之后，数据还是不错，就可以把播放量，涨粉量一起投，100块钱投播放量，100块钱投涨粉量

计划分类 #

出单计划 #

商品购买 + 自定义 + 基础定向 + 快手达人

TIP

出单最快的方式就是找同行，对标账号一般都是找3个月内起号的，作品数量不超过30个，粉丝数是2万到20万之间的

标签计划 #

商品购买/ROI + 自定义 + 基础定向 + 购物意向
商品购买/ROI + 自定义 + 基础定向 + 关键词
商品购买/ROI + 自定义 + 基础定向 + 商业兴趣
商品购买/ROI + 自定义 + 基础定向 + 九大人群

TIP

标签计划的目的是去触达一些有可能买我产品的人群

拉新计划 #

商品购买 + 智能优选 + 基础定向
ROI + 智能优选 + 基础定向

TIP

拉新计划的目的在于，快手是个吃老粉的平台，如果流速非常稳，慢慢就会触达到老粉，一旦触达到老粉，就会形成粉丝复购率太高，粉丝复购率太高，我们就会影响我们的新流量

拉新计划的几种情况及思路

1.多商品的情况
多商品算出价可以按照平均客单价的利润去进行出价，但即使这样操作，也会发现因为平均客单价不代表是你刚好卖的最多的产品销量，这时就会有两种情况，一个是出价过低，出价过低，高客单价的产品，转化不动。另一个是出价过高，出价过高，低客单价的产品去跑量，有可能会造成转化成本过高，超成本，超成本的情况下就可能会造成亏钱，当我们处在这种情况，不太好出价的时候，我们可以把转化目标从商品购买转为ROI

2.降低复购率
在快手小店的电脑端，大数据复盘里面，可以看到复购率是多少，但凡复购率超过80%，那么你的成交基本上都是以老粉为主，老粉的成交是有周期的，不可能一直去薅老粉，而且都是粉丝在购买成交的话，系统也会对应推送，接下来流量主要是以推送粉丝为主，就会减少新流量的进入，这时我们要做拉新，降低复购率，需要去建ROI的拉新计划

3.补人气
直播间人气掉了，需要补人气，补流速的时候，这时也需要建ROI的拉新计划

4.放量
这时也需要建ROI的拉新计划，放量的前提，一个是利润够，一个是出单数量够

拉新计划的ROI系数设置和技巧

多商品的情况下，在理论ROI的基础上上浮1到2

补人气的情况下，一个计划建两个组一起跑，两个组同样的预算，同样的出价。ROI的设置，在实际ROI的基础上下浮1到1.5，日预算选300就可以，不让它跑太快，并且定向只选基础定向就可以，其他的深层定向都不要选

降低复购率的情况下，ROI的系数可以打低一点，把系数打低，是为了触达更多的人群，人群多了，流速上去了，破了层级了，慢慢复购率也就下来了，另外在广告设置中，可以先选优选广告位，如果效果一般，可以选指定广告位的快手单列广告和快手双列广告，如果想复购率降的更快，可以只选快手单列广告。可以加定向也可以不加定向

放量的情况下，第一波放量，要将定向选择设置为智能定向，并打一下基础定向，第二波放量，定向选择设置为智能定向，并且不设基础定向，第三波放量，定向选择设置为自定义定向，但什么都不选

浅层计划 #

粉丝关注/商品点击

TIP

打浅层计划，人群比较泛。因为粉丝比较少，跑量跑不起来，这时我们可以建一些浅层计划，让系统至少有可以探索的人群
加入浅层计划，一般是指的标签打的比较慢，直播间人气比较低，客单价比较高

浅层计划的几种情况和思路

高质量进人，建议出价为0.1，日预算为300，定向最好是打对标同行，人群包打到150万到200万就差不多了，一般是在搞活动或者高客单价的情况下会用

商品点击，建议成本上限为1-1.5，日预算为300，人群包控制在1000万到2000万之间，一般是新号，粉丝很少，投流时探索不到什么人群，或者是高客单价的情况下会用

粉丝关注，一般新号的时候用，还有就是不挂车的直播间可以用，另外就是配合进人想要抬高直播间人气的，最后一个就是高客单价的时候会用，建粉丝关注的时候，人群包最理想的情况是卡在400万到600万之间

计划调节 #

TIP

新手上来先建出单计划+标签计划(商品购买)，每个计划如果3天都出不了10单，那就全部暂停，重新新建，14天之后把计划删掉。能够有一点单了，如果投产不正的话，先暂停计划，调整人货场或者换品，然后用移动金牛 + 商品购买 + 智能优选 + 智能出价 + 小额叠投，投一段时间，如果能投产正了，在进入到PC金牛，建出单计划 + 标签计划 + 拉新计划（商品购买），当这个阶段投产比较理想的时候，后续可以用出单计划 + 拉新计划 (商品购买)，这一步可有可没有，如果你希望在出单的同时人气能高一点，你可以适当的加一条出单计划 + 标签计划 + 拉新计划（ROI），在到后面，用出单计划 + 拉新计划（ROI），在往后用拉新计划 + 放量计划（ROI），在到后面放量计划(ROI)，放量之后，如果说标签有点歪，可以加点出单计划（商品购买）巩固下模型

金牛的底层逻辑 #

金牛竞价机制ECPM = ( 出价 * 预估点击率 * 预估转化率 * 1000 ) + GPM + 互动因子

TIP

ECPM是千次曝光的收益
CPM是千次曝光的花费
GPM是千次观看成交金额

人-主播的能力

金牛根据什么决定跑量的速度?

CTR,CVR->停留时长->出价->预算
金牛根据曝光收费，根据转化加速曝光

金牛投放人群的顺序

粉丝->触达过的人群->陌生流量

金牛调节思路 #

金牛跑的太慢，不消耗

1.调整直播时间段---早上6点-9点，晚上22点以后，相对来说流量没有这么拥挤

2.调整人群包---这个人群包不是指的我们要去万象DNP里面去运算人群，而是广告组界面里调整人群，这个调整人群是指的调整人群的数量

3.调整系数---前期的时候，主要是通过商品购买去跑，为什么跑商品购买，是为了通过小范围的人群能够出单，建立你的用户画像的标签，后续通过ROI去放量，从你的模型不怎么精准到慢慢的精准再到后面去放大你的模型，所以这个调整系数调的是商品购买的系数，商品购买的系数可以一直加到和你的销售价格持平。调整ROI系数，前期一般建议设置5-6左右，它是一个中间系数，既能薅到老粉，又能探索到新粉，但如果你这个系数降到4-5都不跑，那就继续往下降，降到3-4，降到3-4还不跑，那就继续往下降，降到它能够稳定消耗为止。

4.调整广告组的数量和比例---前期我们起步都是建10个组左右，10个组里面可能包含了9个商品购买，一个ROI，但很多人商品购买不出单，不出单就消耗的慢，ROI刚开始又没有标签，跑的又比较慢，所以直播间的人气上不来，如果一直是这种情况，我们可以慢慢增加ROI的比例，比如说我们现在设7个商品购买，3个ROI，因为ROI一般就是两个目的，第一个是可以帮助我们提升人气，第二个是帮助我们去批量的提升投产，提升投产在标签比较稳定的情况下，可以去批量出单，但是在前期没有标签的情况下，它也会去跑，只是它推送进来的人没有那么精准，但是最起码我们直播间先有人气。

5.加入浅层计划---金牛探索的人群一般就是3类，第一类是粉丝，第二类是之前账号触达过的人群，第三类是陌生人群。当你的账号人群跑的不够多的时候，它没有人可以探索，它就不会跑，所以这个时候，一定要加商品点击，加粉丝关注。加浅层计划，跑个1个星期左右，能够帮助账号加速去探索。

金牛跑的太快，控成本

1.调整系数---1天调整不要超过2次

2.调整预算---预算一加，也会加速曝光加速跑量，正常来说，我们把账号的预算，设置在300，500，1000，就够了。在调整预算的过程当中，比如我的预算设为500，等500的预算跑到60%-70%之间的时候，这个时候我们就要加预算了，因为最后的30%，它可能就减少探索，出单量就比较少了。往上加1.2倍的预算，500可以加到600或650，慢慢往上加。

3.拷贝降系数---特别是跑的好的广告组，如果它的投产比较理想，投产理想是指的，它的出价能够大于它的转化成本，比如出价是10块，转化成本是7块，又能出到10单，这种就不要降，让它跑就行，如果你觉得它跑的太快，想让它的转化成本更低，想要利润高一点的话，那就拷贝降系数。比如你原来设的是10，转化成本是8，能出个20单，在这个20单的广告组的基础之上它有模型，所以我们可以去拷贝，拷贝了以后，把它的系数改成9，那有可能转化成本是7.5，这样我们的转化成本就变低了，转化成本变低，意味着订单成本也会变低，订单成本变低就意味着你的利润会增加。

4.暂停广告组---特别是对于憋单的直播间，打单品打爆品的直播间，快速过款的直播间，你暂停广告组基本上没有太大影响。尾货直播间，直播3个小时，金牛在1个小时到2个小时之间，就暂停了，剩下的全是自然流在拉，金牛暂停了以后，直播间人气两三百人还能维持一个小时到两个小时不会掉，因为它一直都有密集出单，所以说官方会给它奖励流量，另外通过金牛，你的投产数据稳定了它也会拉高你的自然流量，这样的话你的人气还是会比较稳。如果说直播间一直都有出单，但是利润比较薄，金牛跑的又比较快，这个时候你可以让它暂停一会，等广告组的投产上来之后再开

5.调整人群---广告组人群如果是跑了两三千万的，只要这个广告组它能跑，你可以把人群打的精准一点。不过这个不是在直播的时候调整，你可以在下播了以后调整。

金牛转化成本高-利润低

1.客单价---不要卖太低的客单价，利润最起码有个10块钱以上，这样能覆盖到金牛的推流成本，不然你肯定是亏钱的

2.过款时间---过款时间不要太长

3.憋单

4.人群不精准---在前期起号的时候，在人货场到位的情况下，通过金牛商品购买打精准对标同行，是你自己去查到的人群画像或者产品，跟你直播间非常类似的直播间，然后在结合短视频引流。到了中期的时候，你的人群稍微精准一点了，一定要有你的直播节奏和直播技巧，还要搞活动。到了下一个阶段，要去降低转化成本的话，商品购买做辅助，主打ROI拉高投产。

5.换品

金牛转化成本低

转化成本低是我们最终要拿到的结果

1.账户已经能出几百单了---30%的ROI+定向，30%的商品购买+定向，20%的ROI+智能投放，20%的ROI+基础定向

2.账户已经能出上千单了---30%的ROI+基础定向，30%的ROI+智能投放，20%的ROI+定向，20%的商品购买+定向

投流必备知识 #

商品购买和ROI的区别

金牛有个5R模型，分为5个层次的人群，且不同层次人群的质量不同
曝光->浅层->深层->成交->复购
商品购买触达的只是成交和复购，ROI都会触达
商品购买根据出价，决定进人的流速和人群质量，但可能会出现超成本的情况，不会过多帮你控成本，它的控成本是根据你的转化来的，转化的越多，转化成本越低，转化不好，转化成本变高，就需要提高出价才能跑的动，就有可能会亏钱
ROI会提高投产，因为它5个层次的人群都会触达。曝光，浅层，深层的人群不代表没有转化，只是转化的意愿相对低于成交和复购，所以ROI会考虑到综合投产

商品购买的出价

利润的30%->利润的50%->零售价的50%->零售价的100%

投产正

实际ROI = 投入产出比 = 产出 / 投入 = 当日GMV/花费
理论ROI = 1/利润率
保本ROI = (1/利润率) / (1-退货率) = 1/利润率+ (0.5 | 1)
实际ROI > 保本ROI才可以放量

日预算

日预算是为了让广告组起量，能够过冷启动期，一个广告组要能够起量，它的要求是72小时内单个广告组能出到10单以上，能够有一个基础的模型，所以我们的预算最好是我们出价的30到50倍，因为要让他出10单
