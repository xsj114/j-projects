投流 #

投流工具 #

粉条

粉条针对的是粉
粉条触达的是整个池子约3亿人，粉丝更泛，购买力相对较弱
花费成本上，粉条对比金牛，金牛成本更高
粉条瞄准的是私域流量，可以大量的去激活已经关注了你的粉丝和兴趣粉丝

金牛

金牛针对的是电商
金牛触达的是平台的购物人群约1亿人，粉丝更精，购买力相对较强
金牛瞄准的是共域流量，激活的都是陌生用户也包括对你有兴趣的用户

金牛也分PC版和移动版
金牛移动版适合的是，已有粉丝，促成交，且已有粉丝纯度高，相对PC金牛更贵
金牛PC端更精细化，拉新能力强，促成交

投流思路 #

TIP

1.最开始先投粉条,通过粉条让我的账户有一些基础标签,让我的短视频能多一些播放量
2.当账户有了一些粉丝或者短视频的基础，已经触达过一些人群了，我们去带货的时候，先用移动金牛去投，在移动金牛投放的时候，前面的一个星期，我们可以根据你的产品所属的类目，如果是大类目，服装，百货，美妆，食品，就可以去直接投放智能的人群，智能的出价，让系统帮跑量，然后有一些基础的出单数据。如果是一些细分类目，很冷门的产品，这时候就不能智能人群了，这时候要打对标,找你的同行。我们通过移动金牛先投一段时间，如果投产比较正，可以用pc金牛去介入。如果移动金牛投产都不正，这时需要调整人货场
3.通过移动金牛的投放，本身有一些出单，有一些模型，然后在通过pc金牛去投，去放量

TIP

1.找到素材模版，定时定点定量的更新作品，每天更新3到5个，更新5到10个，是最好。挂车是3个不挂车，第4个挂，在发3个不挂车，第4个挂

2.粉条测素材，假设每日日更3条，第一天3条一起投，投点赞评论，选择年龄性别即可。每次投50-100块钱，金额没有绝对的标准，但投的越多数据反馈越精准。投完后可以看到点赞数或者评论数反馈比较好的一条作品和第二天的3条作品在一起投放。5天测下来，测出来一个品录的所有段子，比较好的3到5条素材。如果好的素材，你想依据数据指标去参考的话，封面点击率在5-10之间，完播率在20以上，这样的素材会相对好一点。

3.金牛测转化，看素材能不能跑量，建3条计划，每条计划包含测出来的3-5条素材。通过商品购买，选择最大转化，选择年龄性别即可

4.定向测投产，建3条计划，每条计划对应不同的定向，和测过的素材。通过商品购买，选择最大转化，选择人群定向即可

5.计划稳投产，有以下几个阶段
第一阶段，定向加智能探索人群
2条商品购买计划 + 正常出价 + 测过的定向 + 测过的有投产的素材
2条商品购买计划 + 正常出价 + 智能优选 + 测过的有投产的素材
第二阶段，探索人群到稳投产
2条智能优选的商品购买
2条智能优选的ROI计划
第三阶段，稳投产加通投去放量
2条智能优选的ROI计划
2条自定义通投的ROI计划

计划分类 #

出单计划 #

商品购买 + 自定义 + 基础定向 + 快手达人

TIP

出单最快的方式就是找同行，对标账号一般都是找3个月内起号的，作品数量不超过30个，粉丝数是2万到20万之间的

标签计划 #

商品购买/ROI + 自定义 + 基础定向 + 购物意向
商品购买/ROI + 自定义 + 基础定向 + 关键词
商品购买/ROI + 自定义 + 基础定向 + 商业兴趣
商品购买/ROI + 自定义 + 基础定向 + 九大人群

TIP

标签计划的目的是去触达一些有可能买我产品的人群

拉新计划 #

商品购买 + 智能优选 + 基础定向
ROI + 智能优选 + 基础定向

TIP

拉新计划的目的在于，快手是个吃老粉的平台，如果流速非常稳，慢慢就会触达到老粉，一旦触达到老粉，就会形成粉丝复购率太高，粉丝复购率太高，我们就会影响我们的新流量

拉新计划的几种情况及思路

1.多商品的情况
多商品算出价可以按照平均客单价的利润去进行出价，但即使这样操作，也会发现因为平均客单价不代表是你刚好卖的最多的产品销量，这时就会有两种情况，一个是出价过低，出价过低，高客单价的产品，转化不动。另一个是出价过高，出价过高，低客单价的产品去跑量，有可能会造成转化成本过高，超成本，超成本的情况下就可能会造成亏钱，当我们处在这种情况，不太好出价的时候，我们可以把转化目标从商品购买转为ROI

2.降低复购率
在快手小店的电脑端，大数据复盘里面，可以看到复购率是多少，但凡复购率超过80%，那么你的成交基本上都是以老粉为主，老粉的成交是有周期的，不可能一直去薅老粉，而且都是粉丝在购买成交的话，系统也会对应推送，接下来流量主要是以推送粉丝为主，就会减少新流量的进入，这时我们要做拉新，降低复购率，需要去建ROI的拉新计划

3.补人气
直播间人气掉了，需要补人气，补流速的时候，这时也需要建ROI的拉新计划

4.放量
这时也需要建ROI的拉新计划，放量的前提，一个是利润够，一个是出单数量够

拉新计划的ROI系数设置和技巧

多商品的情况下，在理论ROI的基础上上浮1到2

补人气的情况下，一个计划建两个组一起跑，两个组同样的预算，同样的出价。ROI的设置，在实际ROI的基础上下浮1到1.5，日预算选300就可以，不让它跑太快，并且定向只选基础定向就可以，其他的深层定向都不要选

降低复购率的情况下，ROI的系数可以打低一点，把系数打低，是为了触达更多的人群，人群多了，流速上去了，破了层级了，慢慢复购率也就下来了，另外在广告设置中，可以先选优选广告位，如果效果一般，可以选指定广告位的快手单列广告和快手双列广告，如果想复购率降的更快，可以只选快手单列广告。可以加定向也可以不加定向

放量的情况下，第一波放量，要将定向选择设置为智能定向，并打一下基础定向，第二波放量，定向选择设置为智能定向，并且不设基础定向，第三波放量，定向选择设置为自定义定向，但什么都不选

浅层计划 #

粉丝关注/商品点击

TIP

打浅层计划，人群比较泛。因为粉丝比较少，跑量跑不起来，这时我们可以建一些浅层计划，让系统至少有可以探索的人群
加入浅层计划，一般是指的标签打的比较慢，直播间人气比较低，客单价比较高

浅层计划的几种情况和思路

高质量进人，建议出价为0.1，日预算为300，定向最好是打对标同行，人群包打到150万到200万就差不多了，一般是在搞活动或者高客单价的情况下会用

商品点击，建议成本上限为1-1.5，日预算为300，人群包控制在1000万到2000万之间，一般是新号，粉丝很少，投流时探索不到什么人群，或者是高客单价的情况下会用

粉丝关注，一般新号的时候用，还有就是不挂车的直播间可以用，另外就是配合进人想要抬高直播间人气的，最后一个就是高客单价的时候会用

放量计划 #

ROI + 自定义

TIP

一定要在投产正的基础上去放量

计划调节 #

1.出单计划 + 标签计划(商品购买)
2.投产不正的话，先暂停计划，调整人货场或者换品（2000元以内）
用移动金牛 + 商品购买 + 智能优选 + 智能出价 + 小额叠投
3.出单计划 + 标签计划 + 拉新计划（商品购买）
4.出单计划 + 拉新计划 (商品购买)
5.出单计划 + 标签计划 + 拉新计划（ROI）
6.出单计划 + 拉新计划（ROI）
7.拉新计划 + 放量计划（ROI）
8.放量计划(ROI) 注意：模型歪可以+出单计划（商品购买）

金牛的底层逻辑 #

金牛竞价机制ECPM = ( 出价 * 预估点击率 * 预估转化率 * 1000 ) + GPM + 互动因子

TIP

ECPM是千次曝光的收益
CPM是千次曝光的花费
GPM是千次观看成交金额

预估点击率跟什么有关系？

封面-参照大网红
短视频-投素材，素材是需要经过测试的
场景-直播间很热闹，点进去的机率就大

预估转化率跟什么有关系？

人-主播的能力
货-是否是刚需产品，是否具有性价比
场-场景的搭建

金牛根据什么决定跑量的速度?

CTR,CVR->停留时长->出价->预算
金牛根据曝光收费，根据转化加速曝光

金牛投放人群的顺序

粉丝->触达过的人群->陌生流量

投流必备知识 #

商品购买和ROI的区别

金牛有个5R模型，分为5个层次的人群，且不同层次人群的质量不同
曝光->浅层->深层->成交->复购
商品购买触达的只是成交和复购，ROI都会触达
商品购买根据出价，决定进人的流速和人群质量，但可能会出现超成本的情况，不会过多帮你控成本，它的控成本是根据你的转化来的，转化的越多，转化成本越低，转化不好，转化成本变高，就需要提高出价才能跑的动，就有可能会亏钱
ROI会提高投产，因为它5个层次的人群都会触达。曝光，浅层，深层的人群不代表没有转化，只是转化的意愿相对低于成交和复购，所以ROI会考虑到综合投产

商品购买的出价

利润的30%->利润的50%->零售价的50%->零售价的100%

投产正

实际ROI = 投入产出比 = 产出 / 投入 = 当日GMV/花费
理论ROI = 1/利润率
保本ROI = (1/利润率) / (1-退货率) = 1/利润率+ (0.5 | 1)
实际ROI > 保本ROI才可以放量

日预算

日预算是为了让广告组起量，能够过冷启动期，一个广告组要能够起量，它的要求是72小时内单个广告组能出到10单以上，能够有一个基础的模型，所以我们的预算最好是我们出价的30到50倍，因为要让他出10单
