## 利从哪来

::: tip 利益来源的三种方式
被赠与---更多是传承，所谓遗产<br/>

掠夺---抢的隐蔽方式是剥削，剥削就是我占有了生产资料，你来劳动，付出劳动价值，最后产生的剩余价值归我。剥削之所以能存在，前提是核心生产资料被我所占有<br/>

交换---交换的本质逻辑是，拿你有的，换你要的。在交换的核心逻辑当中，第一，专业性是大部分人能用的去交换的底层逻辑。第二，是手里的资源和产品，包括服务。第三，是信用<br/>
:::

## 商业的本质

`商业的本质是满足需求`，这个过程中，我们要理解的是需求的变化，`真需求`和`伪需求`

::: tip 自己的生意，到底满足的是什么需求？
满足`人的`需求，比如猫粮这个产品，是因为猫的主人有更方便喂食自己猫咪的需求，有更多营养化喂食自己猫咪的需求，总之这不是猫的需求，而是人的需求
:::

::: tip 客体分离
你的生意的客户和用户分别是谁?<br/>
`用户`是使用和消费这个产品的人<br/>
`客户`是谁买单，谁是你的客户<br/>
`客户`满意能够成交，`用户`满意能够引发复购和转介绍<br/>
:::

## 商机的本质

商机的本质是满足需求的新方式

老方式：`用（什么方式）满足（谁的）（什么）需求（相对优势）`<br/>
新方式：`用（什么方式）满足（谁的）（什么）需求（新在哪里）`


## 万能成交公式

`成交 = 需求 ✖️ 信任 ✖️ 购买力`

做经营性行为的朋友们，切记每天把你所有的时间，所有的精力，所有的能量，全部用于提升自己被信任的能力上，核心经营的是信任

::: tip 平常应忙在提升客户的信任度上
如何让更多的人信任你<br/>
如何让信任你的人相信你更深<br/>
如何让别人相信你的路径更短<br/>
如何用一些欲望和恐惧的手段来快速地让他为信任买单<br/>
:::

## 成交场景

销售渠道的本质是成交场景

要去意识去找到老成交的新渠道，新渠道本质上就是成交场景的变化，这也是我们能够快速把生意提升一个门槛，或者是扭转生意现状的主要应用场景和手段


## 卖一切都是卖信息

你的产品或服务的品质，和它能不能卖出去没有半毛钱关系<br/>
你的产品或服务的品质，只和他的复购，转介绍，退货率有关<br/>
卖不卖的出去和你会不会卖信息有关
