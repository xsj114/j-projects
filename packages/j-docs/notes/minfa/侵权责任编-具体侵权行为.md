## 承揽人致人损害的责任与承揽人遭受损害的责任

<table>
    <tr>
        <th colspan="3">承揽人致人损害的责任与承揽人遭受损害的责任（《民法典》第1193条）</th>
    </tr>
    <tr>
        <td rowspan="4">法条</td>
        <td>①</td>
        <td>《民法典》第1193条规定：“承揽人在完成工作过程中造成第三人损害或者自已损害的，定作人不承担侵权责任。但是，定作人对定作、指示或者选任有过错的，应当承担相应的责任。”</td>
    </tr>
    <tr>
        <td rowspan="3">②</td>
        <td>《民法典侵权责任编解释（一）》第18条第1款规定：“承揽人在完成工作过程中造成第三人损害的，人民法院依照民法典第一千一百六十五条的规定认定承揽人的民事责任。”（须注意：该款规定有误）</td>
    </tr>
    <tr>
        <td>《民法典侵权责任编解释（一）》第18条第2款规定：“被侵权人合并请求定作人和承揽人承担侵权责任的，依照民法典第一千一百六十五条、第一千一百九十三条的规定，造成损害的承揽人承担侵权人应承担的全部责任；定作人在定作、指示或者选任过错范围内与承揽人共同承担责任，但责任主体实际支付的赔偿费用总和不应超出被侵权人应受偿的损失数额。”</td>
    </tr>
    <tr>
        <td>《民法典侵权责任编解释（一）》第18条第3款规定：“定作人先行支付赔偿费用后，就超过自已相应责任的部分向承揽人追偿的，人民法院应予支持，但双方另有约定的除外。”</td>
    </tr>
    <tr>
        <td rowspan="2">规范内容</td>
        <td>①</td>
        <td>承揽人因承揽“致人”（人身或者财产）损害，成立侵权的：（a）承揽人是否以及如何承担侵权责任，依照法律的规定，既可能是过错责任，也可能是无过错责任【须注意：《民法典侵权责任编解释（一）》第18条第1款的规定有误，因为其规定承揽人只会成立过错责任]。（b）定作人无过错的，定作人不承担责任。（c）定作人（在定作、指示、选任等方面）有过错的，定作人在其过错范围内与承揽人共同承担责任（即“部分连带责任”）；有过错的定作人先行支付的赔偿费用超出其责任范围的，就超出部分，有权向承揽人追偿。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>承揽人因承揽“遭受”（人身或者财产）损害成立侵权的：（a）定作人无过错的，定作人不承担责任；（b）定作人（在定作、指示、选任等方面）有过错的，定作人承担与其过错相应的责任。</td>
    </tr>
    <tr>
        <td>提示</td>
        <td colspan="2">《民法典》第1193条与《民法典侵权责任编解释（一）》第18条的适用范围并不仅仅限于承揽合同，除非法律另有特别规定。《民法典》第1193条与《民法典侵权责任编解释（一）》第18条完全可以类推适用承揽合同之外的其他合同关系。例如，货物承运人、建设工程施工人、受托人、中介人因履行合同义务致人损害成立侵权的，托运人、发包人、委托人是否承担侵权责任，可类推适用《民法典》第1193条与《民法典侵权责任编解释（一）》第18条的规定。</td>
    </tr>
</table>

### 无、限制民事行为能力人致人损害的责任

<table>
    <tr>
        <th colspan="3">1.无、限制民事行为能力人致人损害的责任（《民法典》第1188条）</th>
    </tr>
    <tr>
        <td rowspan="5">法条</td>
        <td rowspan="2">①</td>
        <td>《民法典》第1188条第1款规定：“无民事行为能力人、限制民事行为能力人造成他人损害的，由监护人承担侵权责任。监护人尽到监护职责的，可以减轻其侵权责任。</td>
    </tr>
    <tr>
        <td>《民法典》第1188条第2款规定：“有财产的无民事行为能力人、限制民事行为能力人造成他人损害的，从本人财产中支付赔偿费用；不足部分，由监护人赔偿。</td>
    </tr>
    <tr>
        <td rowspan="3">②</td>
        <td>《民法典侵权责任编解释（一）》第5条第1款规定：“无民事行为能力人、限制民事行为能力人造成他人损害，被侵权人请求监护人承担侵权人应承担的全部责任的，人民法院应予支持，并在判决中明确，赔偿费用可以先从被监护人财产中支付，不足部分由监护人支付。</td>
    </tr>
    <tr>
        <td>《民法典侵权责任编解释（一）》第5条第2款规定：“监护人抗辩主张承担补充责任，或者被侵权人、监护人主张人民法院判令有财产的无民事行为能力人、限制民事行为能力人承担赔偿责任的，人民法院不予支持。”</td>
    </tr>
    <tr>
        <td>《民法典侵权责任编解释（一）》第5条第3款规定：“从被监护人财产中支付赔偿费用的，应当保留被监护人所必需的生活费和完成义务教育所必需的费用。”</td>
    </tr>
    <tr>
        <td rowspan="5">规范内容</td>
        <td colspan="2">无、限制民事行为能力人致人损害，依照法律规定成立侵权的，依照下列规则承担责任：</td>
    </tr>
    <tr>
        <td>①</td>
        <td>由监护人承担无过错的替代责任（“承担侵权人应当承担的全部责任”）；并在判决中明确，赔偿费用可以先从被监护人财产中支付（但应当保留被监护人所必需的生活费和完成义务教育所必需的费用），不足部分由监护人支付（须注意：不能判决由有财产的无、限制民事行为能力人承担侵权损害赔偿责任。有财产的被监护人承担的是“法定补充责任“)。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>未成年子女致人损害成立侵权，父母没有离婚的，被侵权人请求父母共同承担侵权责任的，由父母承担连带责任【《民法典侵权责任编解释（一）》第7条]。</td>
    </tr>
    <tr>
        <td>③</td>
        <td>夫妻离婚后，未成年子女致人损害成立侵权的，被侵权人请求离异夫妻共同承担侵权责任的，由离异后的夫妻承担连带责任。离异夫妻之间的责任份额，可以由双方协议确定；协议不成的，可以根据双方履行监护职责的约定和实际履行情况等确定【《民法典侵权责任编解释（一）》第8条]。</td>
    </tr>
    <tr>
        <td>④</td>
        <td>被侵权人请求监护人承担侵权责任，或者合并请求监护人和（有过错的）受托履行监护职责的人承担侵权责任的，属于“固有的必要共同诉讼”，人民法院应当将无民事行为能力人、限制民事行为能力人列为共同被告【《民法典侵权责任编解释（一）》第4条]。</td>
    </tr>
</table>

    <tr>

## 产品责任

![An image](../../assets/civil_262.png)


<table>
    <tr>
        <th colspan="2">1.产品责任概念的要点</th>
    </tr>
    <tr>
        <td colspan="2">产品责任（产品侵权），指因产品缺陷造成他人人身损害、缺陷产品本身的损害、缺陷产品本身以外的其他财产损害，产品的生产者与销售者应当承担的侵权责任。把握产品责任的概念，应注意以下要点：</td>
    </tr>
    <tr>
        <td rowspan="4">缺陷</td>
        <td>须因产品“缺陷”造成他人损害。所谓“缺陷”，指产品具有危及他人人身、财产安全的不合理危险；产品有保障人体健康和人身、财产安全的国家标准、行业标准的，是指不符合该标准。</td>
    </tr>
    <tr>
        <td>“缺陷”的认定。（a）所谓“不合理的危险”，指不符合人们合理期待的安全。第一，如果按照产品说明使用产品而遭受损害，产品具有不合理的危险；第二，若生产者、销售者已经预见到使用者可能以不合理的方式使用产品，但未对因此产生的危险加以警示，产品也具有不合理的危险。（b）产品虽符合保障人身、财产安全的国家标准、行业标准，但存在不合理危险的，应认定产品具有缺陷；产品不符合保障人身、财产安全国家标准、行业标准，一律认定为产品具有缺陷。</td>
    </tr>
    <tr>
        <td>“缺陷”≠“瑕疵”。“瑕疵”，指不符合约定性，包括物的瑕疵和权利瑕疵。产品仅有“瑕疵”而无“缺陷”，只能主张违约责任。产品既有“瑕疵”又有“缺陷”，并因“缺陷”造成他人损害的，成立加害给付，受害人可择一主张侵权或违约（见例2）。</td>
    </tr>
    <tr>
        <td>产品缺陷包括三类：（a）设计缺陷；（b）制造缺陷；（c）指示或警示缺陷。</td>
    </tr>
    <tr>
        <td rowspan="4">产品</td>
        <td>“产品”，是指经过加工、制作用于销售的产品（《产品质量法》第2条第2款）。对“产品”的这一立法上的定义，字面含义过窄，应当予以扩张解释，作如下理解。</td>
    </tr>
    <tr>
        <td>第一，仅限于“动产”。（a）动产虽因添附成为不动产的重要成分，若因其缺陷致他人损害，仍成立产品责任；（b）“建设工程”不属于产品（建设工程因缺陷致人损害的，适用《民法典》第1252条第1款）；但建设工程使用的建筑材料、建筑购配件和设备属于产品。（c）“信息产品”（书籍、地图、软件、数据），具有不合理危险的（如医学教科书对用药量书写错误，医生因此对患者实施了错误的治疗；再如化学教科书将两种元素的比例标错，学生照此做实验时发生爆炸），原则上，不成立产品责任（以免过度妨害信息自由与言论自由）。</td>
    </tr>
    <tr>
        <td>第二，须为经过“加工、制作”的动产。所谓“加工、制作”，指改变原有物理状态、化学性质的活动。因此，直接来自自然，性质如何完全取决于自然，未经加工、制作的动产，不属于产品。如钓的活鱼、挖的鲜野菜、采摘的鲜蘑菇等。</td>
    </tr>
    <tr>
        <td>第三，须为用于“投人流通”的动产。（a）用于出售、赠与、免费试用、试用买卖均用于“投人流通”。（b）不用于“投人流通”的动产，不属于产品（见例3）。（c）“废品”，如报废汽车、废旧家电，已经退出流通，不再属于产品。</td>
    </tr>
</table>


<table>
    <tr>
        <th colspan="2">2.产品责任的一般规则（《民法典》第1202条至1204条）</th>
    </tr>
    <tr>
        <td>①</td>
        <td>责任主体：生产者与销售者承担无过错责任、不真正连带责任。（a）生产者不仅指最终产品的生产者，还包括有缺陷的原材料、零部件的生产者；产品的进口商视为产品的生产者；生产者不仅包括实际的生产者，还包括表见生产者（准生产者）。所谓“表见生产者”，指在他人生产的产品上标注自已的名称、商标的人。（b）销售者指以销售产品为经营活动的人（偶尔出售二手产品的人，不属于销售者）。销售者包括整个销售链条上的批发商与零售商。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>归责方式：（a）对外：生产者与销售者均承担无过错资任；（b）对内追偿：销售者系过错责任；销售者无过错的，生产者承担全部最终责任（销售者承担责任后有权向生产者全额追偿）；销售者有过错的，销售者承担全部最终责任（生产者承担责任后有权向销售者全额追偿）（《民法典》第1203条）。</td>
    </tr>
    <tr>
        <td>③</td>
        <td>对缺陷的产生具有过错的仓储者、运输者、设计者等第三人不对外承担责任（不能列为被告，充其量列为无独三）。生产者、销售者对外承担产品责任后，有权向有过错的第三人部分（或全额）追偿（《民法典》第1204条）。因为有过错的仓储者、运输者不属于承担产品责任的适格主体。</td>
    </tr>
</table>

<table>
    <tr>
        <th colspan="4">4.违反跟踪观察义务的产品责任（《民法典》第1206条）</th>
    </tr>
    <tr>
        <td rowspan="2">法条</td>
        <td>①</td>
        <td colspan="2">《民法典》第1206条第1款规定：“产品投人流通后发现存在缺陷的，生产者、销售者应当及时采取停止销售、警示、召回等补救措施；未及时采取补救措施或者补效措施不力造成损害扩大的，对扩大的损害也应当承担侵权责任。”</td>
    </tr>
    <tr>
        <td>②</td>
        <td colspan="2">《民法典》第1206条第2款规定：“依据前款规定采取召回措施的，生产者、销售者应当负担被侵权人因此支出的必要费用。”</td>
    </tr>
    <tr>
        <td>例子</td>
        <td colspan="3">甲公司生产A型笔记本电脑，于2022年4月1日投人流通。乙公司销售A型笔记本电脑。2024年4月1日，经鉴定确认，A型电脑存在偶尔丢失数据的缺陷，并且，自2022年4月1日至2024年4月1日期间，因此给数百名用户共造成损失6亿元。</td>
    </tr>
    <tr>
        <td rowspan="4">规范内容</td>
        <td>①</td>
        <td colspan="2">产品投入流通后发现存在缺陷（设计缺陷、制造缺陷、警示缺陷）的，生产者、销售者应当履行“跟踪观察义务”，即及时采取停止销售、警示、召回、跟踪监视等补救措施。须注意：若不履行跟踪观察义务或者履行不适当，则产品具有“跟踪观察缺陷”（前三种缺陷之外的另一种缺陷）。</td>
    </tr>
    <tr>
        <td rowspan="3">②</td>
        <td colspan="2">产品投入流通后发现存在的缺陷，包括两种情形：第一，依照投人流通时的科学技术水平就能认定为缺陷而由于过错没有发现的情形（简称“情形一”）；第二，依照投人流通时的科学技术水平不能认为是缺陷现在构成缺陷的情形（简称“情形二”）。不同情形，生产者、销售者应当承担责任的范围不同，具体而言（结合上面的“例子”叙述)：</td>
    </tr>
    <tr>
        <td>(a)</td>
        <td>对因“情形一”所述缺陷自2022年4月1日至2024年4月1日期间所造成的损害，无论生产者、销售者是否履行跟踪观察义务，生产者、销售者均应依照《民法典》第1203条的规定承担侵权责任。对因“情形一”所述缺陷自2024年4月2日以后造成的损害，分两种情况而不同：第一种情况，若履行了合理的跟踪观察义务（产品即不存在“跟踪观察缺陷”），因不合理使用该产品造成的损害（即“扩大的损害”），生产者、销售者不承担责任。第二种情况，若未履行合理的跟踪观察义务（产品即存在“跟踪观察缺陷”），因使用该产品造成的损害（即“扩大的损害”），生产者、销售者应当承担责任（不真正连带责任）。</td>
    </tr>
    <tr>
        <td>(b)</td>
        <td>对因“情形二”所述缺陷自2022年4月1日至2024年4月1日期间所造成的损害，无论生产者、销售者是否履行跟踪观察义务，生产者、销售者均不承担责任（依照《产品质量法》第41条第2款的规定免责）。对因“情形二”所述缺陷自2024年4月2日以后造成的损害，分两种情况而不同：第一种情况若履行了合理的跟踪观察义务（产品即不存在“跟踪观察缺陷”），因不合理，使用该产品造成的损害（即“扩大的损害”），生产者、销售者不承担责任。第二种情况，若未履行合理的跟踪观察义务（产品即存在“跟踪观察缺陷”），因使用该产品造成的损害（即“扩大的损害”），生产者、销售者应当承担责任（不真正连带责任）。</td>
    </tr>
</table>

<table>
    <tr>
        <th colspan="3">5.就产品责任承担惩罚性赔偿责任的条件（《民法典》第1207条）</th>
    </tr>

        <td>法条</td>
        <td colspan="2">《民法典》第1207条规定：“明知产品存在缺陷仍然生产、销售，或者没有依据前条规定采取有效补救措施，造成他人死亡或者健康严重损害的，被侵权人有权请求相应的惩罚性赔偿。”</td>

    <tr>

        <td colspan="2">符合以下两个条件的，受害人（赔偿权利人）除有权请求生产者、销售者承担补偿性赔偿责任，填补遭受的全部实际损害外，还有权另外主张惩罚性损害赔偿。惩罚性赔偿的标准按以下规则确定：（a）符合法律规定的特别情形的，依照法律的特别规定（如《消费者权益保护法》第55条，《食品安全法》第148条）；（b）不符合法律特别规定的情形的，由裁判者自由裁量确定数额。</td>

        <td>①</td>
        <td>生产者、销售者明知缺陷存在，仍然生产、销售或者生产者、销售者未履行跟踪观察义务并及时采取合理的补救措施致使损失扩大的</td>
    </tr>

        <td>造成受害人死亡或者健康权严重损害的后果。</td>

## 环境污染和生态破坏责任

<table>
    <tr>
        <th colspan="3">1.环境污染和生态破坏责任的概念</th>
    </tr>
    <tr>
        <td>法条</td>
        <td colspan="2">《民法典》第1229条规定：“因污染环境、破坏生态造成他人损害的，侵权人应当承担侵权责任。”</td>

    <tr>
        <td rowspan="3">理解</td>
        <td>① </td>

    </tr>
    <tr>
        <td>③</td>
        <td>所谓“生态破坏”，指人类生活、生产活动引起的生态系统的功能退化和服务减少，因而对人类生存发展以及环境本身发展产生不利影响的现象。</td>
    </tr>

    <tr>
        <td>② </td>
        <td>《生态环境侵权解释》第3条规定：“不动产权利人因经营活动污染环境、破坏生态造成相邻不动产权利人损害，被侵权人请求其承担生态环境侵权责任的，人民法院应予支持。”</td>
    </tr>
    <tr>
        <td rowspan="4">理解</td>
        <td>① </td>
        <td>《生态环境侵权解释》仅适用于“生态环境私益侵权”，指加害行为经由生态环境给特定的民事主体造成人身、财产损害所成立的侵权。</td>
    </tr>
    <tr>
        <td>② </td>
        <td>成立“生态环境私益侵权”的加害行为包括两个类型：（a）污染环境（如《生态环境侵权解释》第1条第1项规定的物质型污染和第2项规定的能量型污染）；（b）破坏生态（如《生态环境侵权解释》第1条第3项与第4项的规定）。</td>
    </tr>
    <tr>
        <td>③ </td>
        <td>污染环境的加害行为，即使合理、合规、合法，仍可成立生态环境私益侵权，多次考到的一种情形是，排污行为“符合国家、地方污染物排放标准”但是造成损害的，仍可成立生态环境私益侵权。</td>
    </tr>
    <tr>
        <td>④</td>
        <td>破坏生态的加害行为，原则上，需要达到不合理、不合法、不合规的程度，才能成立生态环境私益侵权。</td>
    </tr>
</table>


<table>
    <tr>
        <th colspan="3">4.不作为生态环境私益侵权案件处理的法定情形</th>
    </tr>
    <tr>
        <td rowspan="2">法条</td>
        <td>① </td>
        <td>《生态环境侵权解释》第2条第1款规定：“因下列污染环境、破坏生态引发的民事纠纷，不作为生态环境侵权案件处理：（一）未经由大气、水、土壤等生态环境介质，直接造成损害的；（二）在室内、车内等封闭空间内造成损害的；（三）不动产权利人在日常生活中造成相邻不动产权利人损害的；（四）劳动者在职业活动中受到损害的。</td>
    </tr>
    <tr>
        <td>② </td>
        <td>《生态环境侵权解释》第2条第2款规定：“前款规定的情形，依照相关法律规定确定民事责任。”</td>
    </tr>
    <tr>
        <td rowspan="4">理解</td>
        <td>① </td>
        <td>（a）成立生态环境私益侵权，要求加害行为“经由环境介质”“间接”造成“特定人”的人身、财产损害，呈现为如下过程：加害行为作用于环境介质和生态系统→一→环境要素和生态系统发生不利改变（环境要素发生物理、化学、生物或放射性特征的不良改变；生态系统的平衡被打破）一→→因此进一步（间接）造成特定人的人身、财产损害。（b）因此，未经由大气、水、土壤等生态环境介质，直接造成损害的，不成立生态环境私益侵权。（c）依照其他法律的规定，可能成立《民法典》第1165条规定的过错侵权、《民法典》第1236条至第1244条规定的高度危险责任等。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>（a）室内、车内等“封闭空间”不属于“生态环境”的范畴。（b）在室内、车内等封闭空间内，因装饰装修物、电器设备、其他物品遭受损害的，不成立生态环境私益侵权。（c）依照其他法律的规定，可能成立过错侵权、产品侵权、高度危险责任等。</td>
    </tr>
    <tr>
        <td>③</td>
        <td>（a）不动产权利人在日常生活中，释放噪声、烟气、震动、臭气、尘埃、光污染、电磁波辐射等不可量物，造成相邻不动产权利人损害的，不成立生态环境私益侵权。（b）应当依照《民法典》第288条、第289条和第294条关于“相邻关系”的规定处理。</td>
    </tr>
    <tr>
        <td>④</td>
        <td>（a）劳动者在职业活动中受到损害的，不成立生态环境私益侵权。（b）应当依照法律关于“工伤”的特别规定处理。</td>
    </tr>
</table>

## 饲养动物损害责任


<table>
    <tr>
        <th colspan="2">1.概念界定</th>
    </tr>
    <tr>
        <td>① </td>
        <td>须为饲养的动物。不包括微生物（细菌、病毒），也不包括野生动物。培养、保管的微生物致人损害的，一般属于《民法典》第1239条规定的占有或者使用高度危险物致人损害的责任。受保护的野生动物致人损害的，由国家承担赔偿（补偿）责任。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>须为动物“固有的危险实现”。判断是否属于固有危险实现，通说采“动物自主行为说”，如牛抵人、狗咬人、鸡啄人、马踢人、小猪夜卧道路将人倒摔伤等。因此，若非基于动物自主行为致人损害，便不成立饲养动物损害责任：（a）例如：动物自阳台跌落或坠落砸伤人或物。（b）再如：甲出卖给乙的一头牛，患有猪流感，交付后，导致乙的另外3头牛受传染死亡。</td>
    </tr>
    <tr>
        <td>③</td>
        <td>须因动物独立加害。动物在人的驾驭、支配下致人损害（如主人唆使其狗扑咬他人），不成立饲养动物损害责任，属于人（以动物为工具）实施的加害行为。</td>
    </tr>
    <tr>
        <td>④</td>
        <td>既包括直接致人损害，也包括间接致人损害。后者如：狗追咬人，致使被追者跌人水沟受伤；再如：马受惊撞倒路旁的车辆，车辆倾覆致人损害。</td>
    </tr>
</table>
