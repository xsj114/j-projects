    </tr>
    <tr>
        <td>（a）</td>
        <td>财产损害赔偿。赔偿义务人应当对死者的近亲属赔偿“死亡赔偿金”。死亡赔偿金按照受诉法院所在地上一年度“城镇居民人均可支配收入”标准，按二十年计算。但六十周岁以上的，年龄每增加一岁减少一年；七十五周岁以上的，按五年计算。<br/>（《人身损害赔偿解释》第15条)</td>
    </tr>
    <tr>
        <td>（b）</td>
        <td>精神损害赔偿。赔偿义务人应当对死者的近亲属赔偿“精神抚慰金”。</td>
    </tr>
</table>

## 姓名权

<table>
    <tr>
        <th colspan="4">1. 姓名权的客体</th>
    </tr>
    <tr>
        <td rowspan="5">原则</td>
        <td rowspan="5">正式姓名</td>
        <td colspan="2">姓名权的客体是自然人的“正式姓名”（即本名），须符合以下四个条件：</td>
    </tr>
    <tr>
        <td>①</td>
        <td>由姓和名组成。（a）姓（姓氏），用于标表血缘和家族归属；（b）名，用以标表自然人个人（即“区别人已”)。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>属于文字符号和标记，且须使用规范汉字（即简体中文）。</td>
    </tr>
    <tr>
        <td>③</td>
        <td>具有可识别性，即姓名与其标表的自然人建立稳定的对应关系。换言之，相关公众“闻其名，知其人”（当然，本名一般都具备这一特征）。</td>
    </tr>
    <tr>
        <td>④</td>
        <td>须经户口登记机关办理完毕登记（《民法典》第1016条第1款）。</td>
    </tr>
    <tr>
        <td rowspan="4">例外</td>
        <td rowspan="4">其他标志</td>
        <td colspan="2">笔名、艺名、网名、别名（字号、姓名的简称、乳名、绰号、外国人的中文译名）等特定名称符合下列三个条件的，“参照”法律关于姓名权的规定，受同等保护：<br/>（《民法典》第1017条）</td>
    </tr>
    <tr>
        <td>①</td>
        <td>具有一定的知名度，为相关公众所知悉。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>已经与该自然人建立了“稳定的对应关系”（不要求达到“唯一的对应关系”的高度）。</td>
    </tr>
    <tr>
        <td>③</td>
        <td>擅自使用该特定名称的行为，足以产生相关公众在“指代同一性上”发生混淆的可能性。</td>
    </tr>
</table>

## 肖像权与声音权

### 肖像权

<table>
    <tr>
        <th colspan="2">1.肖像权的客体</th>
    </tr>
    <tr>
        <td colspan="2">肖像权的客体为自然人的肖像。<br/>《民法典》第1018条第2款规定：“肖像是通过影像、雕塑、绘画等方式在一定载体上所反映的特定自然人可以被识别的外部形象。”<br/>肖像须具备以下要素：</td>
    </tr>
    <tr>
        <td>①</td>
        <td>须为通过造型方式对自然人全部或者部分外部形象的视觉再现。<br/>（a）肖像并非自然人的身体本身，而是通过影像、雕塑、绘画等造型方式所形成的对自然人外部特征（外部形象）的视觉再现。<br/>（b）须为视觉再现，对自然人外部形象的文字再现、味觉再现等，不成立法律意义上的肖像。<br/>（c）自然人身体的部分所形成具有可识别性的视觉再现，亦可为肖像权的客体（如腿模、手模的腿和手的视觉再现）。
</td>
    </tr>
    <tr>
        <td>②</td>
        <td>须固定在一定载体上。就是说，客观的自然人外部形象→以造型方式制作→形成可识别性的视觉再现→固定于一定载体=肖像权的客体（肖像）</td>
    </tr>
    <tr>
        <td>③</td>
        <td>具有可识别性。即社会一般公众以一般注意力能够识别出其为某特定自然人的外部形象。</td>
    </tr>
</table>

![An image](../../assets/civil_176.jpg)

<table>
    <tr>
        <th colspan="3">2.肖像权的主体与内容·（《民法典》第1018条与第1019条）</th>
    </tr>
    <tr>
        <td rowspan="3">法条</td>
        <td colspan="3">《民法典》第1018条第1款规定：“自然人享有肖像权，有权依法制作、使用、公开或者许可他人使用自己的肖像。”</td>
    </tr>
    <tr>
        <td colspan="2">《民法典》第1019条第1款规定：“任何组织或者个人不得以丑化、污损，或者利用信息技术手段伪造等方式侵害他人的肖像权。未经肖像权人同意，不得制作、使用、公开肖像权人的肖像，但是法律另有规定的除外。”</td>
    </tr>
    <tr>
        <td colspan="2">《民法典》第1019条第2款规定：“未经肖像权人同意，肖像作品权利人不得以发表、复制、发行、出租、展览等方式使用或者公开肖像权人的肖像。”</td>
    </tr>
    <tr>
        <td>主体</td>
        <td colspan="2">自然人</td>
    </tr>
    <tr>
        <td rowspan="4">权能内容</td>
        <td>①</td>
        <td>肖像制作权。即“再现权”，指决定是否通过造型手段通过视觉形式再现个人形象的权利。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>肖像公开权。指决定是否将制作的肖像公之于众的权利。</td>
    </tr>
    <tr>
        <td>③</td>
        <td>肖像使用权。指决定是否利用制作的肖像从事一定活动（自已或者许可他人通过复制、发行、做广告、代言、用于游戏、作为雕塑陈列等方式使用）的权利。</td>
    </tr>
    <tr>
        <td>④</td>
        <td>肖像维护权。指有权禁止他人对自已的肖像实施丑化、污损等行为</td>
    </tr>
</table>

### 个人信息的概念、要素、类型

![An image](../../assets/civil_188.jpg)

<table>
    <tr>
        <th colspan="3">“可识别性”VS“关联性”</th>
    </tr>
    <tr>
        <td rowspan="2">理解</td>
        <td>①</td>
        <td>《民法典》第1034条第2款规定，个人信息须具有“可识别性”。这是从信息本身出发，依信息能否直接或间接识别特定的自然人来判断是否属于个人信息。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>《个人信息保护法》第4条第1款规定，个人信息须具有“关联性”。这是从信息主体出发，依信息是否与已识别或可识别的特定自然人有关来判断是否属于个人信息。“关联说”没有否定“可识别性”，但是，强调个人信息具有相对性，有些信息，对某一处理者而言属于个人信息，但对另一处理者而言则不属于个人信息。</td>
    </tr>
    <tr>
        <td>例子</td>
        <td colspan="2">甲公司雇员李某的工作证号码是RD201603110003。①对甲公司而言，李某属于已识别（或可识别）的自然人，因此，对甲公司而言，李某的工作证号码RD201603110003属于李某的个人信息。②对乙公司而言，若不能通过RD201603110003这一工作证号码识别李某，那么，对乙公司而言，李某的工作证号码RD201603110003就不属于李某的个人信息。</td>
    </tr>
</table>
