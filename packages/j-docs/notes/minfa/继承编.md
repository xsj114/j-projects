        <th colspan="2">2．下列权利（义务），不属于遗产</th>

    <tr>
        <td>①</td>
        <td>被继承人享有的人格权、身份权（有例外：著作权中的发表权可以继承）。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>被继承人死亡后，其亲属应得的死亡赔偿金、抚恤金。</td>
    </tr>
    <tr>
        <td>③</td>
        <td>“承包收益”可以作为遗产，但是，被继承人享有的土地承包经营权不属于遗产（原因有二：第一，承包以“农户”为承包人；第二，以集体土地所有权为基础的土地承包经营权具有“成员权”的性质）。但有两个例外：（a）对林地的土地承包经营权，可以作为遗产继承（《农村土地承包法》第32条第2款）；（b）通过招标、拍卖、公开协商等方式取得的对“四荒地”的土地经营权，可以作为遗产继承（《农村土地承包法》第54条）。</td>
    </tr>
    <tr>
        <td>④</td>
        <td>被继承人享有的国有、集体自然资源用益物权。</td>
    </tr>
    <tr>
        <td>⑤</td>
        <td>被继承人享有的自留地、自留山、宅基地使用权。</td>
    </tr>

##  继承权、受遗赠权的放弃（《民法典》第1124条）

<table>
    <tr>
        <th colspan="3">1.继承权的放弃（《民法典》第1124条第1款）</th>        
    </tr>
    <tr>
        <td>法条</td>        
        <td colspan="2">《民法典》第1124条第1款规定：“继承开始后，继承人放弃继承的，应当在遗产处理前，以书面形式作出放弃继承的表示；没有表示的，视为接受继承。</td>        
    </tr>
    <tr>
        <td>时间</td>
        <td colspan="2">继承人放弃继承权的意思表示，应当在继承开始后，遗产分割前作出【《民法典》第1124条第1款，《民法典继承编解释（一）》第35条】。</td>
    </tr>
    <tr>
        <td rowspan="3">方式</td>
        <td>①</td>
        <td>原则上，应以书面形式向遗产管理人或者其他继承人表示【《民法典继承编解释（一）》第33条】。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>在诉讼中，继承人向人民法院以口头方式表示放弃继承的，要制作笔录，由放弃继承的人签名【《民法典继承编解释（一）》第34条】。</td>
    </tr>
    <tr>
        <td>③</td>
        <td>继承开始后，遗产处理前，继承人没有表示的（“单纯沉默”），视为接受继承（《民法典》第1124条第1款）。换言之，继承人不能以“单纯沉默”的方式放弃继承权。</td>
    </tr>
    <tr>
        <td>限制</td>
        <td colspan="20">继承人因放弃继承权，致其不能履行法定义务的，放弃继承权的行为无效【《民法典继承编解释（一）》第32条】。须注意：下面“特别提示”所述的《民法典婚姻家庭编解释（二)》第11条的规定。</td>
    </tr>
    <tr>
        <td>效力</td>
        <td colspan="20">《民法典继承编解释（一）》第37条规定：“放弃继承的效力，追溯到继承开始的时间。”据此，放弃继承权具有溯及力，溯及到继承开始时（被继承人死亡时），继承人自始不享有继承权。</td>
    </tr>
    <tr>
        <td rowspan="2">撤回</td>
        <td>①</td>
        <td>遗产处理前或者在诉讼进行中，继承人对放弃继承反悔的，由人民法院根据其提出的具体理由，决定是否承认［《民法典继承编解释（一）》第36条第1分句］。</td>
    </tr>
    <tr>
        <td>②</td>
        <td>遗产处理后，继承人对放弃继承反悔的，不予承认［《民法典继承编解释（一）》第36条第2分句]。</td>
    </tr>
</table>
