# j-is

## Install

```
npm i @j-xushijie/j-is --save
```

## Usage

### ESM

```
import is from '@j-xushijie/j-is'
is.array( [] );
```

### COMMONJS


```
const is = require('@j-xushijie/j-is')
is.array( [] );
```
